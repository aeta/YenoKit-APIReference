  • closeBatch()

    Closes this batch. This may fail if conditions are not met as defined in BatchStatusErrors and will throw the corresponding error.

    Important

    Once a batch is closed, the batch is finalized and no changes to its properties can be further made. In addition, mutating methods are disabled and will not make any changes to the batch.

    Precondition

    All guests must be closed.

    Declaration

    Swift

    public mutating func closeBatch() throws
  • addOrder(_:)

    Adds an order to the batch. If preconditions are not met, this will throw the corresponding error as defined in BatchStatusErrors

    Precondition

    Important

    Once an order has been added to the batch, it cannot be removed.
