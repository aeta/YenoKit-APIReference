    This method will attempt to post Object to server. If an Object of ID already exists on server, this method will create a new Object with a new ID. The existing Object of ID will not be overwritten.

    Important

    POST ignores the local ID property and will create a new Object on server regardless of whether or not Object of ID already exists on server.
