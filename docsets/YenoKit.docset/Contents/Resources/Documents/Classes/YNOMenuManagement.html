YNOMenuManagement

public class YNOMenuManagement

A singleton for managing a POS Menu. Integrates with YenoKit-Server.

  • Upon accessing properties, it is required to check whether or not MenuManagement has completed the initial synchronization via hasInitialSyncCompleted. If !hasInitialSyncCompleted, use initialSyncFinishedHandler to execute actions once hasInitialSyncCompleted. See example below: let management = YNOMenuManagement.shared if !management.hasInitialSyncCompleted { management.initialSyncFinishedHandler = { // rit dit dit doo doo ... } }
  • There should only be one instance of this in an app at any given time. Having more than one could cause synchronization issues. The initializer is private to prevent any cases of this.
