Functions

The following functions are available globally.

  • performOn(_:closure:)

    Perform a block on the specified queue. This is just a nicer wrapper around the dispatch_async() Grand Central Dispatch function.

    Declaration

    Swift

    public func performOn(_ queueType: QueueType, closure: @escaping () -> Void)

    Parameters

    queueType

    The queue to execute the block on
    closure

    The block to execute Example usage: performOn(.Main) { self.tableView.reloadData() }

  • delay(_:queueType:closure:)

    Perform a block on a queue after waiting the specified time.

    Declaration

    Swift

    public func delay(_ delay: TimeInterval, queueType: QueueType = .main, closure: @escaping () -> Void)

    Parameters

    delay

    Time to wait in seconds before performing the block
    queueType

    Queue to execute the block on (default is the main queue)
    closure

    Block to execute after the time specified in delay has passed Example usage: // Wait for 200ms then run the block on the main queue delay(0.2) { alert.hide() } // Wait for 1s then run the block on a background queue delay(1.0, queueType: .Background) { alert.hide() }
